# fsn-bootstrap-confidence-intervals
This repository contains R code and data for simulation studies on bootstrap confidence intervals for Rosenthal’s fail-safe number (FSN) in meta-analysis.
The study evaluates different bootstrap methods, including standard normal approximation, percentile, and BCa intervals, under various distributional settings (normal, half-normal, skew-normal, and skew-t).
The code reproduces all simulation results and figures presented in the manuscript.
